Otrzymują Państwo pewien ciąg znaków, w którym należy znaleźć jego środkowy(e) znak(i). Ciąg może zawierać jedno słowo, kilka symboli lub całe zdanie. Jeśli chcą Państwo poćwiczyć z podobnym przypadkiem, proszę spróbować Median misja.
