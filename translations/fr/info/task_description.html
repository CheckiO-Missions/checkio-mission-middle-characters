On vous donne une chaîne de caractères dont vous devez trouver le(s) caractère(s) central(aux). La chaîne peut contenir un mot, quelques symboles ou une phrase entière. Si la longueur de la chaîne est paire, vous devez renvoyer deux caractères du milieu, mais si la longueur est impaire, vous ne devez en renvoyer qu'un seul. Pour plus de détails, consultez l'exemple.

Si vous souhaitez vous entraîner davantage avec un cas similaire, essayez Median mission.

Entrée: Une chaîne de caractères (string).

Sortie: Le(s) caractère(s) du milieu sous forme de chaîne (string).
