You have some string and you want to find the middle characters of it. The string may contain only one word or even few symbols or it can be a whole sentence. If the lenght of the string is even - you should return two middle characters and if the length is odd - only one. More details you can see in Example.

Input: String.

Output: The middle characters.

Example:

middle('example') == 'm'

middle('very-very long sentence') == 'o'
middle('I') == 'I'
middle('no') == 'no'
{% else %} 
middle('example') == 'm'
middle('test') == 'es'    
middle('very-very long sentence') == 'o'
middle('I') == 'I'
middle('no') == 'no'
{% endif %}

How it is used: For work with texts.

Precondition:
1 <= the length of the text <= 100
